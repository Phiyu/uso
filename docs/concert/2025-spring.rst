上半场
~~~~~~

* **卡巴列夫斯基**：D大调简易变奏曲，作品40之1

* **柴可夫斯基**：四季，作品37（六月：船歌）
* **德彪西**：两首阿拉伯风格曲，L.66（第一首：生动的行板）
