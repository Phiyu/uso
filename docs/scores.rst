关于乐谱
========================

请下载祖传谱务手册。

- 使用 IMSLP（国际乐谱库）检索与下载古典乐谱。
- 零基础教学 Sibelius 与 PhotoScore 等软件，涵盖电子识谱与谱面美化排版。
- 基础乐理，提供针对圆号、单簧管、萨克斯等移调乐器的实用转调方法。
- 规范曲谱命名与归档，揭秘乐团打印机使用流程及乐谱分发技巧。
- 内含中、英、德、法四语种乐器名称对照表，以及详尽的乐团活动室（东活7楼 / 西活302）图文导航。
